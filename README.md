# Personal Portfolio Website

A clean, responsive personal portfolio website built from scratch using only **HTML**, **CSS**, and **JavaScript** — no frameworks, no libraries, no dependencies.

---

## 📁 Project Structure

```
portfolio/
├── index.html        # Main HTML file
├── style.css         # All styles and responsive design
└── script.js         # Interactivity and animations
```

---

## ✨ Features

- **Fully responsive** — looks great on desktop, tablet, and mobile
- **Smooth scrolling** navigation between sections
- **Interactive UI** powered by vanilla JavaScript
- **No dependencies** — pure HTML, CSS, and JS only
- **Fast load times** — lightweight with no external frameworks
- **Cross-browser compatible**
